# Portfolio — Eldi Bicari

Portfolio personnel **one-page** réalisé pour le M2 *Cultures et Métiers du Web* (Université Gustave Eiffel).
Site **bilingue FR/EN**, en **HTML, CSS et JavaScript purs** (sans framework), pensé pour GitHub Pages.

## Contenu
Présentation, projets (Noli, SocioSim / MIMESIS, Dico-Labor, Benchmarking AI Chatbots, mémoire de M1, projets de mathématiques),
parcours illustré par une **courbe sigmoïde**, compétences, expériences et contact.

## Stack technique
- HTML5 sémantique, CSS (palette *viridis*), JavaScript vanilla
- Bilingue FR/EN : dictionnaire `francais` / `anglais` indexé par `id` dans [js/main.js](js/main.js)
- Responsive et accessible : lien d'évitement, focus clavier visible, `prefers-reduced-motion`, attributs `alt` / `aria-label`

## Lancer en local
Ouvrir `index.html` dans un navigateur, ou utiliser l'extension **Live Server** de VS Code.
